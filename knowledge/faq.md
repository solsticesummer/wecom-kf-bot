# DramaClaw 客服知识库

> 本知识库来自 DramaClaw 产品使用手册与产品白皮书。客户的问法、用词、语言可能和这里不同，
> 请抓住关键词和意图去匹配，用这里的信息组织出有帮助的回答。

## 关于产品

- 产品名称：DramaClaw（由 Claymore AI Lab 出品）。
- 产品简介：DramaClaw 是一套 AIGC 短漫剧工业化生产平台，面向 AI 短剧、漫剧、小说改编视频、
  解说剧等多类型视频创作。它不只是「生成一个片段」，而是把创意、文本、资产、镜头和成片，组织成
  可迭代、可协作、可规模化的生产流程。
- 一句话理解：先把文本变成可管理的项目，再把角色、场景、道具、声线整理成可复用资产，最后逐集生成
  脚本、镜头、音频、视频，并合成为成片。
- 主线流程：文本 → 资产 → 镜头 → 视频 → 合成。
- 核心价值：降低门槛、提升效率、角色/场景/道具一致性稳定、沉淀长期 IP 资产、支持多人团队协作。
- 一个重要理念：顺序很重要。前面的文案、角色、场景或首帧错了，后面的视频通常会跟着错；先跑通一集
  闭环，再逐步提高画质，不要一开始就盲目反复抽卡生成视频。

## 适用人群和场景

- 适合人群：小说推文 / 解说剧创作者、AI 短漫剧制作团队、短剧工作室、个人创作者，以及 MCN、
  品牌内容营销、文旅 / 非遗 / 教育培训等机构团队。
- 常见用途：解说剧 / 小说改编视频、精品剧 / 仿真人剧、IP 可视化样片、电商产品介绍、品牌广告宣传、
  文旅 / 非遗内容、教育培训视频、游戏 / 虚拟人预演。

## 七大模块（一句话说明）

- 虾料：项目的内容源头，上传 / 粘贴小说或剧本、选择项目类型与风格、生成结构预览。
- 虾塘：项目资产库，管理角色、场景、道具、声线，决定后续生成是否稳定、一致。
- 虾镜：最常用的主工作台，按「脚本 → 镜头 → 合成」推进，完成文案、草图、首帧、音频、视频和导出。
- 虾画：无限画布，适合导演和专业团队自由组合多模态素材、做关键镜头深度精修，确认后写回主线。
- 虾导：AI 导演助手，可查项目进度、检查缺失资产、推进任务、建议下一步。
- 虾格：风格模板，统一项目画风、提示词和避免指令。
- 虾条：任务中心，查看生成任务的状态、进度、日志和错误。

## 两条创作路径

- 主线工作流（推荐新手 / 批量生产）：沿「虾料 → 虾塘 → 虾镜 → 合成」稳定推进，系统把流程固定下来，
  减少「不知道下一步做什么」。适合小白、解说剧、批量生产。
- 无限画布工作流（适合导演 / 专业团队）：在虾画里不受主线顺序约束地自由探索关键镜头，确认后写回主线。
- 正式项目通常混合使用：普通镜头走虾镜，有难度的关键镜头进虾画。可以理解为——主线像「流水线」，
  虾画像「导演工作台」。

## 登录与项目管理

- 打开 DramaClaw 登录页，点击「灵感发生」打开登录弹窗，输入用户名和密码，登录后进入项目管理中心。
- 项目管理中心是所有作品的入口，每个项目是独立空间，包含文本、角色、场景、道具、声线、剧集、镜头、
  视频和合成导出。可按进行中 / 归档 / 回收站查看，可按更新时间或名称排序。
- 建议命名方式：「作品名_批次_负责人」，方便多项目团队快速定位。
- 项目可归档、移到回收站、恢复或永久删除；永久删除前务必确认影响范围。

## 项目共享与权限

团队项目建议先确定分工再分配权限。四种角色：
- 查看者：适合客户、审核人员、非操作成员，仅查看验收，不建议用于制作。
- 编辑者：编剧、美术、分镜、视频操作人员，可参与制作和任务运行。
- 管理员：项目负责人、制片、主创，可管理成员和重要设置。
- 所有者：项目创建者，拥有最高权限。

## 怎么开始用（第一次跑通一集）

第一次不追求每个镜头完美，只要剧情看得懂、角色不明显混乱、能合成导出，就算跑通。先完成闭环，
再逐步提高画质。

1. 新建项目：进入项目管理中心，点击「新建项目」。
2. 导入文本：进入「虾料」，选择项目类型与风格，上传或粘贴剧本，检查结构无误后点击「导入」。
3. 建立资产：进入「虾塘」，自动提取或手动创建角色、场景、道具，并配置声线。
4. 规划剧集：进入「虾镜」，点击「规划剧集」，为目标集规划身份、场景和道具。
5. 生成脚本：在某一集的「脚本」页检查原文 / 分镜文本，必要时用 AI 改写，再生成脚本（Beat）。
6. 制作镜头：在「镜头」页，按文案 → 草图 → 渲染图 / 首帧 → 音频 → 视频的顺序处理每个 Beat。
7. 合成导出：在「合成」页确认所有镜头就绪，选择分辨率和字幕选项，合成成片并下载。

新手建议：先用短文本或一章内容测试，别一开始上传整本超长小说；第一集控制在可管理的 Beat 数，
先跑通再精修；优先保证主角肖像、核心身份、核心场景和默认声线。

## 虾料（导入小说 / 剧本）

- 虾料是项目的内容源头，所有解说剧或精品剧项目都要先建立文本来源。
- 导入步骤：进入「虾料」→ 选择项目类型（精品剧 / 解说剧）与视觉风格（古装、动漫等）和默认人种 →
  选择「上传小说」拖拽文件（支持 txt、md、docx），快速测试可切到「粘贴文本」→ 检查解析的字数、
  段落、章节 → 点击「导入」。
- 项目类型上传后会锁定，如需切换通常要重新导入或新建项目。
- 导入后检查：总字数是否与原文大致一致、检测章节数量和标题是否合理、预计集数是否符合、项目配置是否正确。
- 重新导入是覆盖性操作，会重建图谱，可能影响角色、剧集、脚本、草图、音频、视频等结果；已开始创作的
  正式项目，建议先备份或确认影响范围再重新导入。

## 虾塘（角色 / 场景 / 道具 / 声线资产库）

资产越清楚后面越省事，资产越混乱镜头阶段越容易返工。

**角色管理：** 进入「虾塘 → 角色」，可「自动提取」从图谱识别角色或手动添加；补充姓名、别名、
定位、性别、年龄段、身形、外貌和面部提示词；在肖像区生成或上传肖像并保留满意版本；为不同剧情
阶段 / 服装 / 年龄建立身份/造型（也可在虾镜规划剧集后「规划身份」由 AI 自动识别）；在语音区上传 /
录音 / 裁剪声线。要点：别名要写全（避免同一角色被拆成多人）；面部提示词写年龄、脸型、五官、发型、
气质，不只写服装；服装变化明显时新建身份，不要只改角色描述。

**场景管理：** 进入「虾塘 → 场景」，「从图谱构建」自动提取或「新建场景」；填写名称、类型、环境
描述和叙述描述；核心场景先补齐源图，需要正反打时补背面图，重复出现的地点建议生成 360 全景或导演
世界；同一地点不同状态用「添加场景变体」（plate）表达。

**道具管理：** 进入「虾塘 → 道具」，「新建道具」填写名称、类型、所属角色和视觉提示词；「生成参考图」
或「上传道具」补齐参考图，多个道具可「批量生成参考图」；在镜头文案区选择「出场道具」后，草图、
渲染图、视频才会引用该道具。

**声线管理：** 进入「虾塘 → 声线」。第三人称解说剧通常先配置项目默认解说声线；第一人称解说剧需先
设置解说主角，再到该角色声线管理里配置。声线支持上传音频、浏览器录音和裁剪。声线缺失时批量配音或
角色对白可能无法正常生成。

## 场景变体、导演世界与 3GS（空间一致性）

- 场景变体：表达「同一个地点在不同时间 / 剧情状态下的样子」（如白天 / 夜晚办公室、战后宫殿、雪天
  小镇）。变体挂在基础场景下、继承空间结构、只叠加状态差异，所以始终看起来是同一个地方——比每种
  状态都新建独立场景更不容易穿帮。
- 导演世界：可取景的「片场」，用来固定空间结构、人物位置、道具位置和机位；最终美术质感仍在草图、
  首帧、视频阶段完成。
- 3GS：正面 3GS（基于源图）、背面 3GS（反打 / 换角度）、360 3GS（从全景可环视）、自定义 3GS
  （导入外部 3D / 扫描资产）；可导出控制图 / 导演合成图带回草图或首帧。
- 小白建议：先补齐基础场景源图 →（必要时背面 / 360）→ 打开导演世界 / 3GS → 选目标 Beat 确认空间
  → 放置人物 / 道具 / 占位确认站位 → 调机位导出控制图 → 回虾镜或虾画生成草图 / 首帧 / 视频。

## 虾镜（脚本 / 镜头 / 合成主工作台）

结构是「剧集列表 → 单集工作台」；单集顶部有脚本、镜头、合成三个阶段；镜头内再分文案、草图、
渲染图 / 首帧、音频、视频。

- 剧集规划：没有剧集时点「规划剧集」从章节结构生成；进入某集前建议先规划身份、场景、道具，缺资产
  会影响脚本和镜头。
- 脚本阶段：把本集文本转成可制作的 Beat。解说剧可设目标行数和每行字数后「AI 改写」生成口播稿；
  精品剧可基于原文生成。生成后检查台词 / 解说词是否通顺、说话人是否正确、画面描述、出场身份 /
  场景 / 变体 / 道具是否漏选。
- 镜头阶段：左侧选 Beat，顶部可设草图模型、渲染模型、画幅、视频引擎（切画幅只影响新生成的内容）。
  **务必按顺序处理：文案 → 草图 → 渲染图 / 首帧 → 音频 → 视频。** 可多选 Beat 批量计划。
  - 文案区：编辑台词 / 解说词、类型、场景 / 变体 / 时间、画面描述、出场身份 / 道具；角色或道具不对
    时先来这里检查。
  - 草图区：先确定构图、人物位置、动作大方向，不追求画质；不满意可重生、上传、姿势编辑、裁剪、
    换背景，或打开虾画深入编辑。
  - 渲染图 / 首帧：视频生成的关键参考图，检查背景锚点、角色脸、服装身份、场景、道具、有无多余文字。
  - 视频区：选视频模型（不同模型所需素材不同），设生成模式（新手用默认首帧模式）、时长（单镜头
    4-6 秒更好控）、分辨率（测试用低档、定稿再提高）、画幅（短视频常用 9:16，横屏 16:9），写清
    主体 / 动作 / 运镜 / 禁止项，可用 AI 优化提示词。
- 合成阶段：确认没有未就绪镜头（缺草图 / 音频 / 视频会提示）→ 选分辨率 → 选是否加字幕 →「合成剧集」；
  完成后可预览、下载视频、导出 SRT 或导出 ZIP。

## 虾画（无限画布与精品镜头精修）

- 虾画是自由无限画布，可上传素材、拖入项目资产、组合图片 / 视频 / 音频 / 文本 / 360 / 导演世界 /
  技能节点，做多版本候选比较，适合导演和专业团队做关键镜头精修，也支持团队并行协作。
- 写回主线：虾画里的结果先是「候选」，不会自动覆盖主线；只有明确点「写回」并选好目标槽位（角色 /
  场景 / 道具 / Beat 草图 / 首帧 / 视频）后才进入正式项目资产。写回角色 / 场景 / 道具是全局资产，
  可能影响多个 Beat，写回前先看缩略图和影响范围提示。

## 虾导 / 虾格 / 虾条

- 虾导：AI 导演助手，适合问「当前项目做到哪了、下一步做什么、哪些资产缺失、某集哪些镜头未就绪、
  帮我生成角色肖像或推进某个任务」。可在左侧导航进入，也可在虾画右侧面板结合画布上下文使用。
- 虾格：风格模板，管理项目画风（UI 标签、风格指令、避免指令、风格 Tag、配置 JSON）；可选预设或
  自定义、上传参考图让 AI 分析风格参数，「应用到项目」设为默认风格。
- 虾条：任务中心，查看生成图片 / 视频 / 配音 / 批量 / 合成任务的状态（投递中、队列中、运行中、
  已完成、失败、已取消）、进度、日志和错误；运行中可取消，终态可删除。页面暂时没结果时来这里看任务。

## 团队与个人使用建议

- B 端团队推荐分工：项目负责人（创建 / 进度 / 权限 / 最终质检）、编剧（虾料导入 / AI 改写 / 脚本

  核心场景、默认声线；普通镜头在虾镜完成，关键镜头再进虾画；不要盲目反复抽卡，先判断问题出在文案、
  草图、首帧、音频还是视频提示词。

## 常见问题 / 故障排查

回答这类问题时，先说明可能原因，再给出解决方法，语气亲切。

- 导入后章节不对：多因原文标题不规范或文件格式混乱；整理章节标题后重新导入。
- 角色识别不完整：多因人物别名多或文本信息不足；在角色页手动补充别名和描述。
- 角色长相不稳定：多因肖像、身份图或面部提示词不统一；先在虾塘统一角色资产，再生成镜头。
- 同一场景前后不一致（穿帮）：多因没有建立源图、背面图、360、导演世界或场景变体；在虾塘补齐场景参考，
  同一地点不同状态用场景变体（plate）管理。
- 镜头里道具缺失：多因 Beat 没有选择出场道具或道具参考图缺失；回到文案区选择道具、补齐参考后重生。
- 草图可用但首帧不对：多因渲染参考不足或背景锚点错误；检查身份 / 场景 / 道具参考，必要时进虾画精修。
- 视频动作不符合预期：多因视频提示词不清楚或首帧动作基础不对；先改首帧或提示词，再重新生成视频。
- 合成按钮不可用：多因某些 Beat 缺音频或视频；在合成页点击未就绪的卡片跳回镜头补齐。
- 虾画结果没出现在主流程：因为结果只是候选、尚未写回；在虾画点击「写回」并选择正确的目标槽位。
- 任务一直没有结果：任务可能仍在队列或运行中，或已失败；进入「虾条」查看进度、日志和错误，必要时取消重试。
